Вариант 35. Пивной импортёр "В Зюзю!" 


Компания занимается поставками импортного пива в российские заведения (бары, пабы, брассерии, клубы, магазины). Компания закупает различное пиво в различных пивоварнях по всему миру, отбирая самые лучшие либо самые популярные сорта. Сотрудниками компании являются менеджер по закупкам, менеджер по продажам, бухгалтер, руководитель. Пивной импортер осуществляет закупки у различных компаний. Менеджер по закупкам оформляет договор о закупках между компанией и производителем пива, В договоре указывается дата договора, номенклатура пива, её количество, закупочная цена в долларах за позицию, сумма договора, а также создается план оплат. При оплате договора формируются исходящие платежные документы. Причем один пункт плана оплат может быть оплачен несколькими платежными документами. В документе указывается сумма в S, дата оплаты, подпись бухгалтера. 



Закупленное компанией пиво поступает на различные склады. Когда закупка частично или полностью приходит на склад компании, формируется накладная, в которой указывается какие позиции поступили. Когда все позиции поступают на склад, в все платежи были выплачены, договор считается выполненным.



Обычно, используется ближайший к пивоварне склад (в случае закупки по собственной инициативе), либо ближайший к заведению склад (при закупке по заказу), либо ближайший к вышеуказанным (в случае заполнения наиболее подходящего склада). Закупленное одной партией пиво может поступить частями на разные склады. Поставка, также, может осуществляться с разных складов. 



Менеджер по продажам осуществляет продажи пива различным заведениям по оптовой цене. Оптовая цена рассчитывается как закупочная цена в рублях по текущему курсу * 1.3. Для осуществления поставок менеджер оформляет договор поставок с заведением, указывает в нем необходимые позиции, сумму, а также составляет план поставок в заведение. Оплата всех поставок происходит по 100% предоплате. Для входящего платежа формируется входящий платежный документ, в котором указывается какие пункты плана поставок были оплачены.



Бухгалтер заверяет платежные документы и устанавливает текущий курс доллар - рубль, Каждый месяц руководитель компании получает отчёт о финансовом состоянии организации и принимает решение об осуществлении новых инициативных закупок пива. Руководитель также нанимает и увольняет сотрудников. 



В рамках исполнения договора поставок между компанией и заведением оформляется документ передачи пива. Документ передачи фиксирует факт отгрузки продукции со склада в адрес конкретного заведения и содержит информацию о договоре поставок, пункте плана поставок, дате передачи, складе-отправителе, перечне передаваемых позиций пива, их количестве и фактической стоимости на момент передачи. Передача может осуществляться частично, при этом одному пункту плана поставок может соответствовать несколько документов передачи. После полной передачи всех позиций по пункту плана поставок он считается выполненным. Договор поставок считается выполненным только при условии полной оплаты всех пунктов плана поставок и полной передачи пива заведению. Все документы передачи заверяются ответственным менеджером по продажам и отражаются в системе складского учета.
